# 宏观监测：看边际、结构和证伪

## 一句话

宏观不负责替你挑股票。它用来识别市场正在定价什么、情景能否延续，以及哪些变量会推翻你的组合逻辑。

## 国内：流动性和景气先看结构

- 货币政策看操作利率、净投放/回笼、到期安排与频率；LPR、DR007 等利率看边际变化，不迷信孤立绝对值。
- M2 与社融要拆居民和企业中长期贷款、票据融资等结构，区分真实融资需求与资金空转。
- CPI/PPI、PMI、工业增加值、投资、社零和进出口，重点看同比与环比、行业分项、量价和边际改善。
- 原始发布机构优先。所有日期、口径和修订都要注明，不能把历史观察阈值当永久规律。

## 海外：把数据放进情景

- 利率决议、就业、通胀和收益率曲线都要比较实际值、预期和前值；单次数据不自动改变长期逻辑。
- 收益率曲线要同时看长短端和变化方向。不要只报“倒挂/不倒挂”，要解释它对应增长、通胀、融资成本和资产定价的哪一种情景。
- 判断衰退可用“人、钱、资源”交叉验证：就业与收入、曲线和信贷、工业需求与新订单。多个来源同向才提高置信度；NBER 等确认性指标不能当领先交易信号。

## 用法：定价逻辑 → 约束 → 证伪

```text
市场在交易什么？
→ 该逻辑依赖哪些利率、通胀、供需或政策条件？
→ 影响哪些行业的收入、成本、估值和现金流？
→ 什么数据或事件会把它推翻？
→ 组合是否过度暴露于同一个宏观假设？
```

地缘冲突和政策冲击要先分短期脉冲与长期均衡。分析能源等品种时，要拆品级、炼化/运输能力、进口依赖和国家利益，不能用表面净进出口一把算完。市场急跌时，先按既定估值和仓位计划行动；底部不是猜出来的。
