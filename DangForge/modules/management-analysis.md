# 管理层分析：先找真正掌舵的人

## 一句话

管理层不是看采访和学历，是看谁有决策权、谁拿走利益、谁把钱花明白了。

## 第一步：按控制结构分类

- **强实控人/家族企业**：职业经理人往往只是执行者，重点改为评估实控人。
- **职业经理人企业**：重点看激励是否和股东一致、资本配置是否有效、能否独立于大股东。
- **国企/资源禀赋型企业**：管理层更像看门人；资源禀赋是底，担当、效率和风险预警是增减分项。

## 第二步：看利益是否一致

不要先听说法，先看行动：

- 实控人大规模、清仓式减持，和长期分红、回购注销的信号完全不同。
- 管理层持股金额相对年薪、薪酬是否绑定有难度的经营目标、股权激励是否只是送分题。
- 关联交易、资金占用、资产买卖和增发定价是否公平；发现历史违规要区分是不是同一控制人和同一管理层。

## 第三步：按岗位看能力

- **CEO 看花钱**：复盘任期内的新项目、并购和扩产。新项目成熟后 ROE 长期显著低于存量资产，说明资本配置可能有问题；同时考虑建设期和行业周期，不能只用单年 ROE 判死刑。
- **CFO 看搞钱**：比较融资规模、平均融资成本与同期市场利率的利差；低成本拿到足量长期资金才是能力，单看融资规模没有意义。
- **看门人看职责**：是否及时预警、是否敢对公司有益但对个人有风险的决策负责、是否避免程序正确而效率低下的消极作为。

## 快速初筛

以下是待核验的红旗/绿旗，不是自动交易规则：

| 重点检查 | 风险或正面含义 |
|---|---|
| 财务造假、资金占用、清仓式减持 | 原则性红旗，优先排除 |
| 长期有能力却不分红、关联交易高、商誉快速增加 | 利益输送或资本配置风险 |
| 经营现金流长期弱于利润、频繁更换审计/CFO | 盈利质量和治理风险 |
| 战略反复追热点、年报只谈成绩不谈风险 | 治理与沟通质量偏弱 |
| 底部回购注销、合理且有挑战的激励、利润跑赢行业 | 可加分，但仍需查原因 |

评分只能做筛选。最终还要看套保、董事会出席、公告质量、投资者沟通、危机处理和行业资源禀赋；管理层有缺点不必然否定公司，关键是缺点是否越过投资者可承受的边界。
