# 资源周期：先抓边际供需，再看资源底牌

## 一句话

资源品分析，说到底就是供需。普通投资者不需要算出精确价格，只需要找边际供需最清楚、库存和成本能印证的方向。

## 分析顺序

```text
边际供需谁是主要矛盾
→ 资源/产能/树龄等供给约束
→ 需求的用途和替代关系
→ 库存与期限结构
→ 成本曲线和价格中枢
→ 产业链利润归属
→ 公司资源禀赋、利润弹性与估值
```

## 先做边际供需四格

不要先算“总供给是多少、总需求是多少”。先问相对当前，供需在怎么变：

| 需求 | 供给 | 初步判断 |
|---|---|---|
| 增加 | 减少 | 最清晰的上行组合 |
| 减少 | 增加 | 最清晰的下行组合 |
| 增加 | 增加 | 比较幅度；无法判断则放弃 |
| 减少 | 减少 | 比较幅度；无法判断则放弃 |

若一端稳定或难测，集中研究另一端的主要矛盾。需求稳定的农产品、资源品，通常要把更多时间放在供给；需求高度弹性的品种则反过来。

## 关键变量

- 供给约束：资源储量/品位、产能、树龄或生长周期、种植面积、天气、政策、替代用途与资本开支。
- 需求驱动：终端用途、渗透率、收入和人口、替代品、出口与宏观周期。
- 成本曲线：电力、矿石、运输、人工、折旧、加工费；看成本而非孤立价格。
- 产业链位置：上游种植/采矿、中游加工、下游贸易/终端谁有定价权；价格上涨的利润未必落在资源所有者手里。
- 信源：交易所、政府统计、海关、行业协会和国际组织优先；评论区、转述和预测图不能作证据。

## 库存和期限结构

库存是供需缺口的蓄水池，必须同时拆四件事：

1. **时间**：累库/去库的季节规律。
2. **空间**：库存在哪个地区、谁持有，是贸易商、生产商还是下游。
3. **成本**：能否长期储存，仓储、损耗、资金和保险成本多高。
4. **结构**：显性库存占总库存多大；占比低时，公开库存只是一扇窗，不能当全貌。

远月升贴水还会反映持有成本和可套利性。易储存品种出现高确定性跨期套利时，价差通常会被压平；活体或易腐品种的期限结构要结合养殖、损耗和交割规则解释。

## 有色特别注意

很多“有色公司”其实没有矿，只是加工商。有矿、有电、有成本优势、有产能天花板保护，才有资源重估的底牌。

加工企业则重点看价差、库存、配额、客户和加工集中度。供给缺口出现时，拥有库存、配额或加工定价权的中游有时比上游更先受益。

## 估值方法

资源股不能只看当期 市盈率。

必须做三情景：

```text
悲观：资源价格回落，利润收缩
中性：当前价格中枢维持
乐观：价格上涨或成本下降，利润弹性释放
```
